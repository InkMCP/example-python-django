# Python / Django

Learn how to deploy a Django application with Ink MCP using a simple prompt.

## Tutorial

Full step-by-step tutorial: [docs.ml.ink/examples/backend/python-django](https://docs.ml.ink/examples/backend/python-django)

### Prerequisites

- An [Ink](https://ml.ink) account
- Ink MCP connected to your AI agent ([setup guide](https://docs.ml.ink/quick-start))

### Quick Deploy

1. Connect Ink MCP to your agent
2. Prompt: "Create a Django REST API for a blog with posts and comments. Use Django REST Framework."
3. Prompt: "Deploy with Ink"
